
Below are the list of tables in DBUtility database which stores SQL Server metadata-

tbl_SQLVersion					-> 	Stores SQL Version details(Edition, Verions, Sevice Pack level, etc.)
tbl_SQLServerConfigs			-> 	Stores SQL Server configuration details(SQL Min/Max memory, MAXDOP value, etc.)
tbl_ServerOSDetails 			-> 	Stores OS related details(Hostname, OS Name, OS Version, IP Address, Domain name, etc.)
tbl_DiskDetails					-> 	Stores Server's disk details(Volume label, total space, free space, etc.)
tbl_SQLSvcAccounts				-> 	Stores Service Account details which are being used for running SQL Server services
tbl_DatabaseBackupDetails		-> 	Stores Database backup details
tbl_DatabaseDetails  			->	Stores Database details(DBName, Collation, Size, Created Data, etc.)
tbl_DatabaseFileSpace			->	Stores Database data and log file details (.mdf,.ndf,.ldf)	
tbl_FailedSQLJobs				-> 	Stores list of failed SQL Jobs
tbl_SysAdmins 					-> 	Stores list of logins having Sysadmin privileges in SQL Server
